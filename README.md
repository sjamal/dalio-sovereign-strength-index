# Sovereign Strength Index Engine 📈
[![License: MIT](https://shields.io)](https://opensource.org)

An open-source quantitative analysis platform implementing Ray Dalio's macrocycle frameworks detailed in *"Principles for Dealing with the Changing World Order"*. It features decoupled ingestion pipelines tracking sovereign performance arrays for global superpowers, combined with adaptive presentation layers optimized for human developers and local AI environments.

## Features Matrix
* **Decoupled Architecture**: Modular ingestion vectors isolation World Bank and FRED indicators data.
* **Algorithmic Cycle Evaluator**: Equal-weighted normalization scoring historical engine paths.
* **Multi-Channel Interface Layers**: Native FastAPI endpoints, a responsive Streamlit UI, and an MCP (Model Context Protocol) Server for AI agent interoperability.
