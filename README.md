# Sovereign Strength Index Engine 📈
[![License: MIT](https://shields.io)](https://opensource.org)

An open-source quantitative analysis platform implementing Ray Dalio's macrocycle frameworks detailed in *"Principles for Dealing with the Changing World Order"*. It features decoupled ingestion pipelines tracking sovereign performance arrays for global superpowers, combined with adaptive presentation layers optimized for human developers and local AI environments.

## Features Matrix
* **Decoupled Architecture**: Modular ingestion vectors isolation World Bank and FRED indicators data.
* **Algorithmic Cycle Evaluator**: Equal-weighted normalization scoring historical engine paths.
* **Multi-Channel Interface Layers**: Native FastAPI endpoints, a responsive Streamlit UI, and an MCP (Model Context Protocol) Server for AI agent interoperability.

### 🔧 Automated Infrastructure & Test Frameworks
*   **Chaos Engine Validation (`tests/test_chaos.py`)**: Continuously injects invalid datatypes, null parameters, and malformed strings to ensure execution calculations remain stable under external API crashes.
*   **Performance Tracking Reporter (`scripts/mcp_bench_reporter.py`)**: Runs high-speed benchmarking loops over JSON-RPC streams to verify communication layers and auto-compile plain-text analytics summaries.
*   **Demographic Contraction Vector (`engine_demographics.py`)**: Integrates Peter Zeihan's country metrics to track labor force shortages alongside Dalio's leverage variables.

## 🏛️ Theoretical Appendix: The Macro Reality of Dalio's Big Cycles

Users reviewing shorter-term historical timelines (e.g., 2010–2026) will observe that the calculated sovereign lifecycle stages remain static (United States locked in Stage 5, China locked in Stage 4). This behavior is mathematically deliberate.

In Ray Dalio's *Principles for Dealing with the Changing World Order* framework, a full "Big Cycle" transpires over **200 to 250 years**. Short-term windows represent single frames at the tail-end of a multi-century regime. 

*   **The United States (Stage 5)** represents an empire experiencing high financial leverage coupled with global overextension. The model locks it here due to structural Debt-to-GDP parameters exceeding 115% combined with significant military commitments.
*   **China (Stage 4)** represents the ascendant competitor empire. It has advanced past productivity rebuilding (Stage 3) into a phase of rapid infrastructure accumulation and leverage growth, positioning it firmly within the "Height of Excesses" parameters.
